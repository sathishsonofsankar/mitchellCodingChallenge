Vehicle Rest Service Api - created for Mitchell International Coding Challenge

Technology Stack:
Apache Maven
Spring Boot
Jersey
Embedded InMemory MongoDB
Rest-Assured

To Build the project:
mvn clean install -DskipTests

To run the applciation:
java -jar target\VehicleRestService-0.0.1-SNAPSHOT.jar

To run Testcases:
mvn clean test
Note:Application has to be up and running for the testcases to function properly
